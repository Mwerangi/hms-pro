Option 1 Top Progress Bar (NProgress)
A thin, animated purple bar that appears at the top of the page during navigation. Automatically enabled for all internal links.
  • Lightweight (8KB)
  • Smooth animations
  • Works with browser back/forward
  • No configuration needed
Triggers on Link Click
Auto-enabled! Just click any link and see the purple bar at the top.
Option 2 Button Loading States
Shows a spinner inside the button and disables it during form submission. Prevents double-clicks and provides clear visual feedback.
  • Prevents double submissions
  • Clear visual feedback
  • Works with any button
  • Auto or manual control
<form class="form-loading">
  <button type="submit">Submit</button>
</form>
// That's it! Automatic loading on submit
Option 3 Full Page Loader
A semi-transparent overlay with a centered spinner for heavy operations like bulk processing, large file uploads, or complex calculations.
  • Full screen coverage
  • Blocks all interactions
  • Clear "processing" state
  • Backdrop blur effect
showPageLoader();
// Do your heavy operation
await processLargeFile();
hidePageLoader();
